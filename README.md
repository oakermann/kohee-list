# KOHEE LIST

KOHEE LIST is a Cloudflare Pages + Workers + D1 cafe curation product
repository. OAP is maintained separately.

Authoritative product docs:

- [Master context](docs/KOHEE_MASTER_CONTEXT.md)
- [Product constitution](docs/KOHEE_CONSTITUTION.md)
- [Roadmap](docs/KOHEE_ROADMAP.md)
- [Execution log](docs/KOHEE_EXECUTION_LOG.md)

Core product rule: public `/data` may expose only approved, non-deleted cafes.
CSV, submissions, review notes, evidence, and internal fields must not bypass
admin approval or leak into public data.
